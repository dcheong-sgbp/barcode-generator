# Barcode Generator

A barcode generator: type any text or number and download a scannable Code 128 barcode as PNG, all in the browser. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/barcode-generator)**

```html
<script src="barcode-generator.js"></script>
<barcode-generator></barcode-generator>
```

## What it does

Code 128 (type B) is the compact, high-density 1D barcode used for shipping, inventory and asset tags. It encodes letters, digits and common symbols, and the generated image includes the mandatory checksum so any standard scanner reads it.

## Install

```bash
npm install @sgbp/barcode-generator
```

or copy `barcode-generator.js` into your project.
